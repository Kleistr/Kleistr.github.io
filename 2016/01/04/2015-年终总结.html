2015 年终总结

Posted on 2016-01-04 |

其实应该算是期末总结，毕竟是从这个学期开始算起的，只有半年而已，姑且就把它当作年终总结吧。

2015注定是不寻常的一年，特别是月开始，ISIS一次又一次地主演着恐怖事件，最震撼的一次是巴黎。最近一次是土耳其击落俄罗斯飞机，此事也跟恐怖主义占了点边。“世界最大黑客组织” Anonymous 向 IS 宣战。这一年，习主席与马英九握手了；这一年，恐怖主义再掀狂潮，甚是嚣张；这一年，美团与大众点评合并；这一年，Apple发布了iPad Pro,12寸Macbook；这一年，太多太多……

成就

  1. 较为系统的学习了C语言，虽然感觉自己还在初级阶段
  2. 入门Python和Html & CSS
  3. 在中国MOOC获得了C语言的证书
  4. 用 AppleScript 完成部分的自动化日常工作
  5. 土木的课程设计，独立完成一栋男生寝室楼的设计工作，已出图

项目

  1. 利用几天时间将个人博客搭建在SAE上，折腾了一阵子主题后，总算满意了
  2. 用C语言编写了一个设计楼梯参数的程序，后续准备添加绘图功能
  3. 在一个获取天气的Python脚本基础上添加一项功能：实时检测天气情况，如若天气非晴则给制定邮箱发邮件提醒
  4. 跟晏哥合作完成了一个拾取斗鱼某直播间弹幕转成语音实时播报的项目

反思

  1. 没能将课余时间利用好，看电影、看直播、看游戏浪费了太多时间
  2. C语言学习深度还不够，只停在初级阶段，还应多练手，多学习
  3. 计划好的学Git，一直拖到年末也没学，明年第一件事就是学习Git，而且制定的计划一定要按时执行
  4. 以为只要用Mac就不会受游戏干扰，好好学习编程语言了，没想到本专业的Proe、3DMax、Revit都是只有win版的，虽然虚拟机可以运行，可一到渲染就……
    5.考试总是让人高兴不起来的事情呢

那些书

除了几本技术书外，还看了几本非技术书，我一向都爱看书的，只是最近感觉阅读量少了好多。

  • 《从你的全世界路过》
  • 《分成两半的子爵》
  • 《第七天》
  • 《追风筝的人》
  • 《鲁南小城的故事》
  • 《蒙古国空前绝后四百年》
  • 《偷影子的人》
  • 《1894，悲情李鸿章》
  • 回看《悟空传》

2016 ToDo List

1. 认真学好C语言，至少做两个项目，拿得出手的项目

2. 学习Python

3. 系统的学习算法与数据结构，并在博客上写学习笔记

4. 读20本书，非技术书

5. 希望找个小项目做
